## Overview
This project focuses on exploring and analyzing sales data from a fictional superstore. 
The goal is to clean, inspect, and visualize the dataset to extract meaningful insights about sales trends, profits, and regional performance.  
This project focuses on analyzing retail sales data using Python and Power BI.


Key tasks in this project include:

- Data cleaning and preprocessing  
- Feature engineering in python 
- Exploratory Data Analysis (EDA)  
- Data visualization with matplotlib, Power bi and ...
- Saving a cleaned version of the dataset  

---

## Dataset
The dataset used in this project contains sales records for a superstore, including information such as:

- `Order ID`  
- `Order Date` and `Ship Date`  
- `Customer Name` and `Segment`  
- `Region`  
- `Product Category` and `Sub-Category`  
- `Sales`, `Profit`, `Discount`  

**Source:** [Superstore Dataset CSV](https://gist.githubusercontent.com/nnbphuong/38db511db14542f3ba9ef16e69d3814c/raw/Superstore.csv)  

---

## Features / Columns
| Column        | Description |
|---------------|-------------|
| Order ID      | Unique identifier for each order |
| Order Date    | Date the order was placed |
| Ship Date     | Date the order was shipped |
| Customer Name | Name of the customer |
| Segment       | Customer segment (Consumer, Corporate, Home Office) |
| Region        | Sales region (e.g., West, East) |
| Category      | Product category |
| Sub-Category  | Product sub-category |
| Sales         | Total sales amount |
| Profit        | Profit earned on the order |
| Discount      | Discount applied on the order |
